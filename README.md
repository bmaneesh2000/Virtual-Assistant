# Virtual-Assistant-PYQT5

An chatbot implemented using bag of words and neural networks byTensorFlow has also other features like presenting content from Wikipedia, accessing websites easier,sending SMS or an E-mail,Saving Notes and Contacts etc by utilizing vaious modules like Chrome web driver,Beautiful soupe,SMTP and other APIs like yandex translator,open weather etc..
